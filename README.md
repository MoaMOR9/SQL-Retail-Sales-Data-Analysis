#Retail Sales Data Analysis (MySQL Project)

This project is a complete MySQL analytics project built using a  retail dataset containing 500 sales transactions. 

Key SQL Analyses Performed

Total revenue

Daily revenue trends

Monthly revenue summary

Sales by product category

Spending by gender

Average transaction value

Top 10 highest-spending customers

Units sold per category

Average customer age by category



Example Insights

Electronics and Home Decor tend to have the highest revenue per transaction.

Female customers show slightly higher total spending overall.

Weekends generate more transactions than weekdays.

The highest revenue spike occurs mid-February (seasonal demand).
